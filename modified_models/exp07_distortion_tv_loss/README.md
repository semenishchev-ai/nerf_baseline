# EXP-07: Distortion Regularization (TV-style)

**Источник:** На основе torch-ngp с идеями из Plenoxels и других работ
**Офиц. реализация:** Отсутствует (авторская модификация)

## Идея
Добавление регуляризации на **дисторсию** (distortion) для улучшения качества реконструкции.

**Ключевая идея:**
- В оригинальном NeRF дисторсия вычисляется как часть loss
- В этой модификации добавляется дополнительная TV-регуляризация на дисторсию
- Это заставляет модель генерировать более гладкие и стабильные результаты

## Архитектурное изменение
- В `train_nerf` добавлен параметр `distortion_tv_weight`
- Добавлен `distortion_tv_loss` в общий loss
- TV-регуляризация применяется к дисторсии по всем уровням

## Файлы
- `network.py` — без изменений (используется стандартный GridEncoder)
- `main_nerf.py` — обучение с distortion_tv_weight

## Пример использования
```bash
python main_nerf.py data/lego \
    --workspace logs/exp07_distortion_tv \
    --iters 10000 \
    --distortion_tv_weight 0.01
```
