# Итерация 1: Модификация архитектуры Neural Radiance Fields (NeRF)

В данном разделе задокументированы все основные этапы и эксперименты, проведённые с целью улучшения результатов оригинального бэйзлайна (`torch-ngp`) без значительной потери в скорости обучения.

## 1. Контекст и генерация гипотез
Взяв за точку отсчета репозиторий `torch-ngp` (бейзлайн на базе Instant-NGP), был сформирован шорт-лист из 7 перспективных архитектурных подходов, базирующихся на новейших исследованиях (статьи 2023–2026 годов), нацеленных на снижение "алиасинга" (aliasing) и проблемы коллизий в Multi-Resolution Hash Encodings. 

## 2. Изолированная имплементация 7 экспериментов
Для строгого сравнительного контроля (ablation study) были разработаны и внедрены 7 независимых модификаций, каждая из которых инкапсулирована в своей папке и наследовала логику оригинального бэйзлайна:

1. **EXP-01: Spatially-Adaptive Hash Masking** — Обучаемые пространственные маски (WACV 2025).
2. **EXP-02: Rotated Hash Encoding (R-MHE)** — Декорреляция коллизий Learnable матрицами поворота координат (ICLR 2026).
3. **EXP-03: Frequency Regularization** — Штраф за рассинхронизацию частот сетки (ECCV 2024).
4. **EXP-04: Hybrid Positional + Hash Encoding** — Комбинирование гармоник для грубых уровней и хеш-сеток для тонких (WACV 2024).
5. **EXP-05: Enhanced MLP** — Увеличенная ширина MLP (64 → 128), skip-connections и GELU.
6. **EXP-06: Scale-Aware Weighting** — Distance-based сглаживание уровней (Anti-Aliasing из Zip-NeRF, ICCV 2023).
7. **EXP-07: Distortion + TV Loss** — Регуляризация прозрачности для удаления "флоатеров" (Mip-NeRF 360).

## 3. Бенчмарки (10k итераций)
Для справедливой оценки каждый из 7 экспериментов прошел процедуру 10,000 итерационного обучения на Synthetic Lego Dataset в идентичных аппаратных условиях:
- **EXP-05 (+0.27 dB)** и **EXP-06 (+0.14 dB)** показали себя как бесспорные лидеры, устойчиво улучшая качество реконструкции без ущерба для скорости.
- Подходы, нацеленные на сглаживание/TV-регуляризацию (EXP-03, EXP-07) значительно просели в PSNR, так как "замылили" высокочастотные детали. Коллизионная маскировка (EXP-01) и гибридный positional (EXP-04) усложнили лосс-ландшафт и не превзошли мгновенную сходимость базового Instant-NGP. 

Все собранные детальные результаты по метрикам занесены в `EXPERIMENTS_LOG.md`. 

## 4. Сборка победителя (new_model_v1)
Взяв лучшие модули из лидерских экспериментов, была синтезирована структура `new_model_v1`, бесшовно объединившая в себе:
- Утолщенную, способную ухватить сложные зависимости MLP с residual-связями (из **EXP-05**).
- Эвристику защиты от алиасинга при помощи `log_scales` и весового дисконтирования сеток (из **EXP-06**).

## 5. Долгосрочное дообучение и проверка потолка (30k)
Для подтверждения гипотезы о том, что `new_model_v1` не просто быстрее сходится, но и обладает более высоким потолком качества:
Был осуществлен прогон `baseline_ngp` и `new_model_v1` на 30,000 итераций.
- Разработанный парсер логов сгенерировал сравнительную кривую `Validation PSNR` (см. `./logs/comparative_psnr_plot.png`). 
- На пик-стадии `new_model_v1` стабильно закрепилась поверх бэйзлайна, достигнув **34.40 dB** PSNR на тестовой выборке по сравнению с 34.31 dB у оригинала, при этом полностью сохранив скорость инференса. 

Метрики финального прогона занесены в файл `results.md`.
